# composant utilisé:

-1 carte electronique OTTO MKS
-4 servomoteur 9g
-1 capteur HC-SR04
-1 batterie rechargable de 9v
-1 cable

# Conception et assemblage du robot:

Impression 3D des pièces
Les différentes pièces du robot ont été imprimées en 3D afin de créer sa structure mécanique.
Assemblage mécanique
Les servomoteurs ont été installés dans les jambes et les hanches pour permettre les mouvements du robot.
Câblage électronique
Les composants électroniques ont été reliés à la carte OTTO-MKS.


# Programmation du robot

Programmation avec Arduino IDE
La programmation du robot a été réalisée avec Arduino IDE afin de contrôler les mouvements et les capteurs.
Fonctions programmées:

- Marche avant ;
- Rotation gauche ;
- Rotation droite ;
- Mouvement des hanches ;
- Détection des obstacles.
